# GCSE Exam Planner 2026

A fully self-contained, static website for planning GCSE exams in Summer 2026.

## Features

- ✅ **Complete exam timetable** — AQA, Edexcel & OCR dates for all major GCSE subjects
- 🎯 **Subject picker** — tick your subjects, instantly see your personal timetable
- 📅 **Calendar view** — visual month-by-month calendar with your exams highlighted
- 📥 **Download `.ics`** — one-click calendar file for Google Calendar, Apple Calendar, Outlook etc.
- 🖨️ **Print timetable** — clean print stylesheet included
- 🔗 **Share link** — shareable URL encodes your subject selection — no login needed
- ⏱️ **Countdown** — days to your next exam

## Deploying to adamtest.site

Place this folder at `gcsetimetable/` in the root of the adamtest.site repo.

The planner will be accessible at: **https://adamtest.site/gcsetimetable**

```
adamtest.site/
└── gcsetimetable/
    ├── index.html
    ├── style.css
    ├── app.js
    └── README.md
```

## Files

```
index.html   — Main page
style.css    — All styles
app.js       — Exam data, logic, ICS generation
README.md    — This file
```

## Important Notice

> Dates are sourced from official AQA, Edexcel and OCR timetables for Summer 2026. Always confirm your personal timetable with your school. Your school's timetable is the definitive source for room numbers, exact start times, and any local arrangements.

Official board timetables:
- [AQA](https://www.aqa.org.uk/exams-administration/dates-and-timetables)
- [Edexcel](https://qualifications.pearson.com/en/support/exam-timetables.html)
